Green ≤ 70% · amber 70–90% · red > 90% (continuous loading guidance — class typically expects sea-load carried with one generator in reserve; verify against your spec and class rules). A source's % uses (rated kW × qty run) vs the mode total.

Method: per consumer and mode, Load(kW) = Rated kW × No. in use × LF × DF. LF = mechanical load factor of the motor/consumer (≈0.8 typical, 1.0 if rated absorbed power entered). DF = diversity/duty factor for the operating mode (0 = not used, 1 = continuous). Set "Sets" to the installed quantity and "In use" per mode (e.g. 2 installed pumps, 1 running). This mirrors common shipyard load-balance practice; classification societies do not prescribe one fixed format — submit per your project specification.
